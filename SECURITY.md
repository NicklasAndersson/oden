# Security Policy

## English
If you discover a security issue, please report it by opening a GitHub issue:
https://github.com/NicklasAndersson/oden/issues/new

Include a clear description, steps to reproduce, expected impact, and any relevant logs or screenshots. Avoid sharing sensitive information. If you need to share sensitive details, use GitHub Security Advisories instead: https://github.com/NicklasAndersson/oden/security/advisories/new

## Svenska
Om du hittar en säkerhetsbrist, rapportera den genom att skapa ett GitHub-ärende (issue):
https://github.com/NicklasAndersson/oden/issues/new

Beskriv problemet tydligt, hur det kan återskapas, förväntad påverkan och relevanta loggar eller skärmbilder. Undvik att dela känslig information. Om du behöver dela känsliga detaljer, använd GitHub Security Advisories istället: https://github.com/NicklasAndersson/oden/security/advisories/new
