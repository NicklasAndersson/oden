# Security Policy

## English
If you discover a security issue, please report it privately via GitHub Security Advisories:
https://github.com/NicklasAndersson/oden/security/advisories/new

For non-sensitive security questions or discussions, you can open a GitHub issue:
https://github.com/NicklasAndersson/oden/issues/new

Include a clear description, steps to reproduce, expected impact, and any relevant logs or screenshots. Avoid sharing sensitive information.

## Svenska
Om du hittar en säkerhetsbrist, rapportera den privat via GitHub Security Advisories:
https://github.com/NicklasAndersson/oden/security/advisories/new

För icke-känsliga säkerhetsfrågor eller diskussioner kan du skapa ett GitHub-ärende (issue):
https://github.com/NicklasAndersson/oden/issues/new

Beskriv problemet tydligt, hur det kan återskapas, förväntad påverkan och relevanta loggar eller skärmbilder. Undvik att dela känslig information.
